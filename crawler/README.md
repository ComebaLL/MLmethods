# Карточка дата-сета с сайта [PoEDB](https://poedb.tw)

1. PoEDB картинки вешей(оружия, гемы, броня и т.д)

Коллекция 2D-иконок всех уникальных предметов из игры Path of Exile. Датасет включает изображения оружия, доспехов, бижутерии, флаконов и камней умений, структурированных по категориям на основе данных сообщества PoEDB.

2. https://poedb.tw/us/Items (База данных PoEDB).

3. Дата сбора 25.02.2026

4. Структура

- Изображения: Папка /images/ с файлами в форматах .webp.
- Метаданные: Папка /raw_html/ содержит исходные страницы категорий, из которых были извлечены ссылки.
- Файл состояния: state.json — реестр всех скачанных объектов с их URL-адресами и статусом загрузки.
- Именование файлов: {Название_предмета}_{Уникальный_хэш}.{расширение}.

5. Характериски данных

- Количество объектов: Ориентировочно 2000 уникальных единиц.
- Язык: Английский (в названиях файлов и атрибутах Alt).
- Наличие пропусков: Практически отсутствует. Если у предмета нет иконки, используется стандартный плейсхолдер игры.
- Разрешение изображений:
    - Стандартные иконки (кольца, камни): от 78x78 px.
    - Крупные предметы (двуручное оружие, доспехи): до 156x234 px.
- Тип данных: Изображения с прозрачным фоном (в большинстве случаев) или на темном фоне игрового инвентаря.

6. Описание полей\атрибутов 
- Item Name (из имени файла): Название предмета, очищенное от спецсимволов.
- Unique Hash: 6-значный идентификатор, созданный на основе URL, предотвращающий перезапись предметов с одинаковыми именами.
- Source URL (в state.json): Прямая ссылка на изображение на серверах GGG (Content Delivery Network).

7. Примеры изображений

- Starforge_Infernal_Sword_a1b2c3.webp
- Tabula_Rasa_Simple_Robe_f4e5d6.webp
- Headhunter_Leather_Belt_z9y8x7.webp

8. Ключевые слова дата-сета

- Game Assets, Path of Exile, Computer Vision, Item Classification, RPG, Icon Dataset.

9. Лицензия и условия использования

- Правообладатель: Все графические ассеты (иконки предметов) принадлежат компании Grinding Gear Games (GGG).
- Условия: Данные собираются в соответствии с GGG Fan Content Policy.
    - Разрешено: Использование для некоммерческих проектов, фанатских сайтов, инструментов для сообщества и обучения нейросетей в личных целях.
    - Запрещено: Коммерческая перепродажа изображений или использование их в других играх.
