# Шпаргалка по regex

![regex](/regexcrib.jpg)

примеры:

- дата дд.мм.гггг: \b\d{2}\.\d{2}\.\d{4}\b

- символ с заглавной буквы: \b[A-ZА-Я][a-zа-я]*\b

- Поиск слова: \bPlayStation\b

- Поиск списка слов: \b(Xiaomi|Apple|Mail.Ru)\b

- Телефон (?:\+7|8)\s*\(?\d{3}\)?[\s-]*\d{3}[\s-]*\d{2}[\s-]*\d{2}
  - `(?:\+7|8)` - номер начинается с +7 или 8
  - `\s*` - 0 или больше пробелов
  - `\(?` - необязательная открывающая скобка `(`
  - `\d{3}` - ровно 3 цифры (код оператора)
  - `\)?` - необязательная закрывающая скобка `)`
  - `[\s-]*` - 0 или больше пробелов или дефисов
  - `\d{3}` - следующие 3 цифры
  - `[\s-]*\d{2}` - разделитель и 2 цифры
  - `[\s-]*\d{2}` - разделитель и последние 2 цифры

- Email \b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}\b

- IPv4 \b(?:25[0-5]|2[0-4]\d|1\d{2}|[1-9]?\d)(?:\.(?:25[0-5]|2[0-4]\d|1\d{2}|[1-9]?\d)){3}\b
  - `\b` - граница слова в начале и в конце
  - `(?:25[0-5]|2[0-4]\d|1\d{2}|[1-9]?\d)` - один октет (число от 0 до 255):
    - `25[0-5]` - числа 250-255
    - `2[0-4]\d` - числа 200-249
    - `1\d{2}` - числа 100-199
    - `[1-9]?\d` - числа 0-99 (без лишних ведущих нулей, кроме 0)
  - `(?:\.(...)){3}` - точка и еще 3 таких же октета
  - `\.` - экранированная точка (именно символ `.`, а не "любой символ")


- URLs \bhttps?:\/\/(?:[A-Za-z0-9-]+\.)+[A-Za-z]{2,}(?:\/[^\s]*)?\b
  - `\b` - граница слова
  - `https?` - `http` или `https`
  - `:\/\/` - последовательность `://`
  - `(?:[A-Za-z0-9-]+\.)+` - одна или несколько доменных частей (`www.`, `site.` и т.д.)
  - `[A-Za-z]{2,}` - доменная зона (`ru`, `com`, `org`, ...)
  - `(?:\/[^\s]*)?` - необязательный путь после домена (`/news/1?x=2`)
  - `\b` - граница слова в конце


- HTML теги <\/?[A-Za-z][A-Za-z0-9]*(?:\s+[A-Za-z_:][-A-Za-z0-9_:.]*(?:\s*=\s*(?:"[^"]*"|'[^']*'|[^\s"'=<> + "" + ]+))?)\s/?>`
  - `<` - начало тега
  - `\/?` - необязательный `/` (для закрывающих тегов)
  - `[A-Za-z][A-Za-z0-9]*` - имя тега (`div`, `a`, `img`)
  - `(?: ... )*` - блок атрибутов, который может повторяться много раз
  - `\s+[A-Za-z_:][-A-Za-z0-9_:.]*` - пробел и имя атрибута
  - `(?:\s*=\s*(...))?` - необязательное присваивание значения атрибуту
  - `"[^"]*"` - значение в двойных кавычках
  - `'[^']*'` - значение в одинарных кавычках
  - `[^\s"'=<>`]+` - значение без кавычек
  - `\s*\/?>` - конец тега, включая вариант `/>` для самозакрывающихся
